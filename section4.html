Le web et internet c’est pareil :

Le WORLD WIDE WEB, système hypertexte public sur Internet. Le Web a été inventé par Tim Berners-Lee et Robert Cailliau des années après Internet. Ce n’est qu’une des applications d’Internet qui permet de consulter, avec un navigateur, des pages reliées entre elles grâce à des hyperliens.

  • Accéder à des documents textuels.
  • Situés sur des machines accessibles par Internet.
  • Reliés entre eux par un mécanisme de lien « hypertexte ».
  • L’actualité .
  • De différentes natures .

Uniform Resource Locator:

Un lien hypertexte est un lien qui permet d’ouvrir une nouvelle page. URL est le nom unique de la page, décrivant l’emplacement d’une ressource. elle contient :

  • L’adresse du serveur.
  • L’adresse de la page sur le serveur.
  1. URL : Pour accéder a un ensemble d’information sur elle serveur .
  2. HTTP : Système client-serveur .
  3. HTML : Le physique d’un site .

Une page Web, c’est un document multimédia, d’écrit élément par élément ,avec un langage de description

  • Stocké dans un fichier
  • Une page Web est interprétée et affichée par un navigateur et est localisée sur Internet à l’aide d’URL.
  • Permet d’accéder à d’autres pages en suivant des liens

Un site web est un regroupement de pages Web autour d’une thématique commune.

  • Emanant d’une même entité accessibles à partir d’une page d’accueil et d’une UR de base communes
  • En général, localisées sur un même serveur
  • Structuré sous forme d’arborescence de dossiers et de fichiers .
  • Permet de définir des chemins relatifs.
HyperText Markup Language Langage de description de pages web

Un document HTML est :

  • Un fichier texte, construit suivant une syntaxe précise.
  • Structuré par des balises.
  • Interprété dans un navigateur.
  • Créé dans un éditeur de pages web ou tout éditeur de texte.

Les principes de langage HTML:

Un fichier HTML doit comporter au minimum ces 4 balises :

  • <html> ... </html>
  • <head> ... </head>
  • <title> ... </title>
  • <body> ... </body>

Les titres et paragraphes :

  • <hn> ... <hn> : Titre de niveau n :1 à 6. Exemple <h1> Grand titre </h1>
  • <p> ... <p> : paragraphe
